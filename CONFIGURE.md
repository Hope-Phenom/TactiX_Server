# CONFIGURE

程序启动前请确保已在操作系统中设置以下环境变量。

| 变量名                   | 值                                                     | 说明                           |
| ------------------------ | ------------------------------------------------------ | ------------------------------ |
| TACTIX_FORUM_USERNAME    | 你的scboy论坛名                                        | 论坛用户名，用于爬取新闻信息   |
| TACTIX_FORUM_PASSWORD    | 你的scboy论坛密码                                      | 论坛账户密码，用于爬取新闻信息 |
| TACTIX_CONNCTION_STRINGS | 服务器连接字符串，包括server、User、Password、database | 用于服务端的数据库访问         |
